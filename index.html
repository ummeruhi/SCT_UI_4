Annotated UX Decisions

Why this upgraded flow is better

  • Added a splash screen to simulate a real mobile app entry point.
  • Short onboarding with skip option reduces friction for returning users.
  • Profile setup collects only essential data first, keeping the flow smooth.
  • Fitness level is separated from goals so recommendations can be more precise.
  • Goal selection uses multi-select because users often have more than one objective.
  • Workout preference step improves personalization for home, gym, yoga, or cardio users.
  • Notification permission is placed near the end, after value is already established.
  • Summary screen confirms all choices and gives closure to the onboarding journey.
  • Progress bar reduces uncertainty and helps users understand how much is left.
  • This prototype remains low-fidelity but is still clickable and realistic for submission.
